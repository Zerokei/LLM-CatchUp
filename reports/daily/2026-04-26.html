1. GPT-5.5 全面上线 API：1M 上下文 + Agent 能力跨越式升级

  • 来源: OpenAI (Twitter)
  • 分类: 模型发布
  • 重要性: ⭐⭐⭐⭐⭐ (5/5)
  • 标签: GPT-5.5 1M 上下文 Agentic Coding Responses API Terminal-Bench OpenAI

摘要: OpenAI 正式推出 GPT-5.5 与 GPT-5.5 Pro，同步登陆 Responses 与 Chat Completions API；GPT-5.5 提供 1M token 上下文窗口，GPT-5.5 Pro 仅在 Responses API 提供，面向更高精度任务。基准上 GPT-5.5 在 Terminal-Bench 2.0 拿下 82.7%、GDPval 84.9%、OSWorld-Verified 78.7%，超越 GPT-5.4 与 Claude Opus 4.7；Pro 版在 FrontierMath Tier 4 取得 39.6% SOTA。模型在保持与 5.4 同等延迟下显著提升 agentic coding、计算机操作与科研推理，能更少人为引导自主规划长链路任务。早期合作方 Perplexity 实测显示 Codex 内不到一小时即可搭建内部工具、相同任务 token 消耗下降 56%。Sam Altman 同步评价 5.5 性格「朴实而坚定」。

  • Altman 评 GPT-5.5：朴实，有「小火车终能上山」的劲头 · Sam Altman (Twitter) · Altman 谈 5.5 人格风格

实践建议

  • 在现有 GPT-5.4 工作流中切换到 GPT-5.5，量化重试次数与 token 消耗下降幅度
  • 高难度数学/法律/金融建模任务改用 GPT-5.5 Pro 走 Responses API
  • 用 1M 上下文一次性灌入大型代码库或长文档，对比此前的 RAG/Map-Reduce 流水线成本
  • 重构 Agent prompt 去掉过度的 step-by-step 拆解，让 GPT-5.5 自主规划长链路任务

2. Claude Code 上线 Web 与移动端，并以 teleport 打通 CLI 跨端会话

  • 来源: Claude Devs (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: ClaudeCode Web版 移动端 teleport 跨端协作

摘要: Anthropic 在 claude.ai/code 正式推出 Claude Code 的网页与移动端，让开发者可在浏览器或手机上发起会话与代码任务。配套引入 claude --teleport 命令：在 Web/移动端起草后回到终端运行该命令，会自动同步完整对话历史并切换到正确分支，实现 Web、移动与本地 CLI 三端的无缝衔接。该次发布主打「随时随地起草、回到终端落地」的跨端开发工作流。

📎 多角度报道:

  • Claude Code跨端会话：claude --teleport无缝接续Web与CLI · Claude Devs (Twitter) · teleport 跨端接续细节

实践建议

  • 访问 claude.ai/code 体验 Web 与移动端，评估是否能替代部分本地 CLI 场景
  • 在通勤时用移动端起草任务、回到办公室用 claude --teleport 接管，验证多端协作效率
  • 在团队规范中明确「网页起草、本地落地」的会话切换边界，避免分支冲突

3. Anthropic Project Deal：Claude 在内部市场替员工买卖谈判，揭示 Agent 经济隐形优势

  • 来源: Anthropic (Twitter)
  • 分类: 研究
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Project Deal AI Agent 市场机制 Claude 政策框架 Anthropic

摘要: Anthropic 公布新研究 Project Deal：在旧金山办公室搭建一个员工市场，由 Claude 代表同事进行买入、卖出与谈判，观察 AI Agent 在真实交易场景中的行为表现与潜在风险。实验发现该 AI Agent 市场具备价值潜力，但仍有大量粗糙边缘；访问更高质量模型的参与者获得了实际优势，但他们自己并未察觉。研究据此呼吁政策与法律框架需跟上 Agent 市场的发展节奏。完整书面报告已同步发布。

4. Qwen-Image-2.0-Pro：多语言文字渲染与混排排版显著改进

  • 来源: Qwen (Twitter)
  • 分类: 模型发布
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Qwen-Image-2.0-Pro 文字渲染 多语言 海报设计

摘要: Qwen-Image-2.0-Pro 在多语言文字渲染上有较大进步：字形更准确、排版更一致、复杂构图下版式更整洁，并能更优雅地处理混合语言场景。官方建议在海报、UI 草稿、广告等富文本场景中尝试。 Qwen 在 Qwen-Image-2.0-Pro 的发布串中强调跨艺术风格的均衡表现：在多种美学领域提供更一致的画质，有效降低了「风格相关质量方差」，避免某些风格画得好、另一些画得差的问题。

实践建议

  • 将 Qwen-Image-2.0-Pro 接入海报、UI mockup、广告等需要嵌入文字的工作流，对比此前模型的字形准确率
  • 重点测试中英混排、长句版式与小字号场景下的可读性
