## 文章详情

### 1. [OpenAI 应对 Axios 开发者工具供应链攻击](https://openai.com/index/axios-developer-tool-compromise/)

- **来源**: OpenAI Blog
- **分类**: 政策与安全
- **重要性**: ⭐⭐⭐⭐ (4/5)
- **标签**: `OpenAI` `供应链安全` `Axios` `macOS` `网络攻击`

**摘要**: 黑客于 3 月 31 日劫持某开发者账号并向 Axios 库发布两次恶意更新，OpenAI 用于签署 macOS 应用的 GitHub 工作流下载了该恶意版本。经调查确认，没有用户数据、知识产权或软件被篡改，攻击被认为与朝鲜关联实体有关。OpenAI 已启动证书轮换，并要求所有 macOS 用户在 5 月 8 日前更新应用，旧版本届时将停止支持。

### 2. [澳大利亚政府与 Anthropic 签署 AI 合作谅解备忘录](https://www.anthropic.com/news/australia-MOU)

- **来源**: Anthropic Blog
- **分类**: 政策与安全
- **重要性**: ⭐⭐⭐ (3/5)
- **标签**: `Anthropic` `澳大利亚` `AI安全` `政府合作` `MOU`

**摘要**: Anthropic CEO Dario Amodei 赴堪培拉与澳大利亚总理签署谅解备忘录，双方将在 AI 安全研究、联合评估和学术合作上展开系统性合作。Anthropic 向澳大利亚国立大学等四所机构提供合计 300 万澳元的 Claude API 额度，并计划在悉尼开设办公室、探索当地数据中心投资，这是 Anthropic 国际扩张战略的重要一步。

### 3. [OpenAI Codex 推出按需计费定价方案](https://openai.com/index/codex-flexible-pricing-for-teams/)

- **来源**: OpenAI Blog
- **分类**: 产品与功能
