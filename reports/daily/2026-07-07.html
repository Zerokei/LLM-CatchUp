1. EdgeBench 发现智能体环境学习缩放律

  • 来源: ByteDance Seed
  • 分类: 研究
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: EdgeBench 智能体评测 环境学习 缩放律 ByteDance Seed

摘要: 字节 Seed 发布 EdgeBench，用 134 个真实长时任务评估智能体如何通过环境反馈持续学习，任务覆盖科学与机器学习、软件工程、专业知识工作、组合优化、形式数学和交互模拟等六类。研究基于约 38,000 小时交互发现，智能体的环境学习曲线高度符合 log-sigmoid 形态，平均 R² 达 0.998。跨代模型对比还显示前沿模型的环境学习速度大约每三个月翻倍，项目已开源 51 个任务和评测框架。

2. GPT-Realtime-2.1-mini 上线：工具调用 + 延迟优化

  • 来源: OpenAI Devs (Twitter)
  • 分类: 模型发布
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: GPT-Realtime-2.1-mini Realtime API 语音模型 工具调用 延迟优化

摘要: OpenAI 在 Realtime API 中上线 GPT-Realtime-2.1-mini，把推理和工具调用带到 mini 实时语音模型，且保持与 GPT-Realtime-mini 相同成本。同一更新还通过改进缓存让 Realtime 语音模型 p95 延迟至少降低 25%。这对语音客服、实时教练和低延迟代理场景更直接地改善了可用性与成本可控性。 OpenAI Devs 表示，通过改进缓存，Realtime 语音模型的 p95 延迟至少降低 25%。这是一条与同线程 GPT-Realtime-2.1-mini 发布配套的性能更新，主要影响实时语音应用的响应速度和交互流畅度。

实践建议

  • 用现有 Realtime-mini 场景做 A/B 测试，重点观察工具调用成功率、p95 延迟和中断恢复。
  • 优先在低风险语音客服、实时教练或语音代理原型中试用新模型。
  • 检查模型别名和计费配置，确认同成本迁移不会误用更高价模型。
