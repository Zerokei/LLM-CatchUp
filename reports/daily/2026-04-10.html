3. Gemini API 推出 Flex 和 Priority 推理层级

  • 来源: Google AI Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Google Gemini API 推理优化 定价 开发者工具

摘要: Google 为 Gemini API 引入 Flex 和 Priority 两个新推理层级，让开发者在成本和延迟之间灵活取舍。Flex 提供更低成本，Priority 保证更低延迟。

实践建议

  • 如果你在使用 Gemini API，切换到 Flex 层级测试批量任务的成本节省效果
