  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: xAI Grok CLI Agent 自动化

摘要: xAI 宣布 Grok Build 进入 Beta，面向所有 SuperGrok 与 X Premium+ 订阅用户开放。功能包括 Plan Mode 规划工作流、用 Imagine 生成图像与视频、以及通过 CLI 构建自动化与 orchestrator。用户可访问 x.ai/cli 上手。

实践建议

  • 若已订阅 SuperGrok 或 X Premium+，访问 x.ai/cli 试用 Grok Build 的 Plan Mode，对比其与 Claude Code、Codex CLI 在多步任务规划上的差异。
  • 将 Imagine 的图像/视频生成接入 CLI orchestrator，搭建「文本计划 → 多模态产物」的本地工作流原型，评估其在内容生产场景的可用性。

2. Qwen3.7-Max 上线隐式缓存：零配置自动降本

  • 来源: Qwen (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Qwen Prompt Caching API 优化 阿里云 推理成本

摘要: 阿里 Qwen 宣布 Qwen3.7-Max 已支持隐式缓存（implicit caching），无需任何配置即可自动启用，开箱即可获得更快响应与更低成本。对于需要更高、更确定命中率的场景，官方建议改用显式缓存（explicit caching），并提供了 Model Studio 的最佳实践文档。

实践建议

  • 在 Qwen3.7-Max 的现有调用上直接观测首 token 延迟与计费变化，验证隐式缓存的真实命中收益，无需改代码。
  • 对系统提示长且稳定的场景（RAG、Agent 工具描述、长 few-shot），按官方最佳实践切换到显式缓存，锁定更高命中率并降低长上下文成本。

3. Anthropic 在梵蒂冈：Chris Olah 回应教皇 AI 通谕

  • 来源: Anthropic Blog
  • 分类: 教程与观点
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Anthropic AI 伦理 可解释性 Chris Olah 梵蒂冈通谕

摘要: 教皇利奥十四世于 5 月 25 日发布关于人工智能的通谕「Magnifica humanitas：在人工智能时代守护人」，Anthropic 联合创始人 Chris Olah 受邀出席发布会并致辞，官方账号同步放出演讲全文。Olah 坦言每个前沿实验室都受到与「做正确的事」相冲突的商业、地缘与个人激励，因此独立于这些激励之外、坚持安全立场的知情批评者至关重要；他强调 AI 系统不像桥梁被工程化设计，而是基于类脑结构在海量人类文本上「生长」出来的，对训练者本身仍保留神秘。他提出三个最需教会发声的议题：对全球穷人的责任（潜在大规模失业且收益集中于少数富国）、对人类繁荣的道德想象、以及对模型内部状态（功能性的喜悦/恐惧/悲伤、类似神经科学的结构、内省迹象）本质的辨析。
