1. Codex/GPT-5.6 加速：700 万周活 + 成本下降

  • 来源: OpenAI Devs (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐⭐ (5/5)
  • 标签: Codex GPT-5.6 Sol ChatGPT Work 代理工作流 开发者工具 成本效率

摘要: OpenAI 围绕 Codex 和 GPT-5.6 展示了一组同日进展：Codex 周活超过 700 万，两个月内发布 150 多项更新，覆盖 GPT-5.6、Ultra、/goal 并行工作、AppShots、Inline edits、Sites、移动端和 SSH 工作流。Sam Altman 进一步称 Codex 与 ChatGPT Work 活跃用户合计达到 800 万，代理类产品一周使用量增长 2.5 倍。GPT-5.6 Sol 被描述为在许多任务中以约一半价格和约两倍 token 效率完成同类工作，但需求激增也可能带来短期服务波动。Notion 的模型评测命令行工具和 Build Week 直播补充展示了它在真实开发、评测和从想法到构建流程中的落地方式。

实践建议

  • 用现有代理任务回放对比 GPT-5.6 Sol 的成功率、输出 token 和端到端成本。
  • 检查 Codex 工作流中可并行拆分的任务，试用 /goal 承载多线开发。
  • 评估 AppShots、Sites、Inline edits 与移动/SSH 工作流是否能减少从原型到 PR 的手工步骤。
  • 为团队建立按任务完成成本而非单 token 价格的评测表。

2. Meta 多模态推理模型亚物奥理论满分

  • 来源: Meta AI (Twitter)
  • 分类: 研究
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Meta AI 物理奥林匹克 多模态推理 评测 科学推理

摘要: Meta AI 称其提交的模型参加亚洲物理奥林匹克理论考试，获得 30/30 满分，并与前三名学生选手并列。该结果被用来展示 Meta AI 模型的高级推理与多模态能力，但推文未披露模型名称、评测设置和可复现实验细节。

3. Claude for Teachers 上线：免费高级能力 + 课程标准

  • 来源: Claude (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude for Teachers K-12 教育 教学技能 课程标准 FERPA 数据隐私

摘要: Anthropic 推出 Claude for Teachers，为美国已验证的 K-12 教师免费提供高级 Claude 能力、教学技能库，并连接基于证据的课程资源和全美 50 个州的学术标准。产品还接入 Learning Commons 等教育工具，并把 Claude Code 与 Cowork 纳入教师工作流，用于班级数据分析、差异化教学材料生成和重复任务调度。Anthropic 同时强调对话不会用于训练模型，学生信息受符合 FERPA 的数据处理协议保护，并配套发布教师 AI 素养课程和开源教学技能库。

📎 多角度报道:

  • Claude for Teachers 强调 K-12 隐私 · Claude (Twitter) · 隐私与产品细节

实践建议

  • 让教师先用标准对齐的课程计划和分层练习做小范围试点。
  • 将 Learning Commons 连接器与现有课程资源比对，检查生成材料是否符合州标准。
  • 为班级数据分析任务设置最小必要数据输入和人工复核流程。

4. Google Images 25 周年：搜索加入图像生成

  • 来源: Google AI Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Google Images 视觉搜索 AI Overviews 图像生成 Nano Banana

摘要: Google 借 Google Images 25 周年回顾视觉搜索演进，并推出两个新能力：面向美国英文桌面用户的可浏览动态图片首页，以及在 Search 的 AI Overviews 中直接生成图像。图像生成由最新 Nano Banana 模型驱动，将在未来数周面向支持 AI Mode 图像创建的地区以英文逐步推出。

实践建议

  • 关注 AI Overviews 图像生成上线地区和入口，测试搜索意图到生成图像的转化质量。
  • 对依赖图片搜索流量的内容站点，评估动态首页和生成式搜索对发现路径的影响。

5. OpenAI 建议按有用工作衡量 AI 投资

  • 来源: OpenAI Blog
  • 分类: 教程与观点
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: OpenAI 代理工作流 AI 投资 成本治理 ChatGPT Work

摘要: OpenAI 提出企业在代理时代不应只看 token 单价，而应衡量每美元带来的有用工作，包括任务完成、节省时间、决策改进和可规模化流程。文章建议从用量与支出可视化、按结果评估模型 ROI、先治理再扩张、投资可复用工作流，以及按已验证需求匹配容量五个方面管理 AI 投资。文中还称 GPT-5.6 在编码代理指数上以更少输出 token 和更短任务时间延续了降本趋势。
