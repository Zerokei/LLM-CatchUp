### 1. [OpenAI Devs × Modal：为长时运行 Agent 准备可规模化的受控工作空间](https://x.com/OpenAIDevs/status/2045561701010202826)

- **来源**: OpenAI Devs (Twitter)
- **分类**: 产品与功能
- **重要性**: ⭐⭐ (2/5)
- **标签**: `Agents SDK` `Modal` `长时运行 Agent` `受控工作空间` `基础设施`

**摘要**: OpenAI Devs 强调：运行代码的 Agent 需要在任务启动时就有一个准备就绪的受控工作空间。合作伙伴 Modal 撰文分享了为何规模化基础设施对使用 Agents SDK 构建的长时运行 Agent 至关重要，指向受控执行环境与状态持久化两个关键议题。

> **实践建议**
> - 阅读 Modal 的博文，了解其为 Agents SDK 提供的受控执行环境方案（隔离、预热、快速启动等能力）。
> - 在自家 Agent 项目中规划长时任务时，优先评估沙箱隔离、状态持久化与冷启动延迟三项基础能力，避免把工作空间问题留到规模化阶段再解决。
