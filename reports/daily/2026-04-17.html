1. Introducing Claude Opus 4.7

  • 来源: Anthropic Blog | 📡 也被 Claude (Twitter) 报道
  • 分类: 模型发布
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude Opus 4.7 Anthropic 模型更新 长任务 指令遵从

摘要: Anthropic 推出 Claude Opus 4.7，是迄今最强的 Opus 系列模型。新版本在长时运行任务上更为严谨，可以更精确地遵循指令，并在汇报前自动验证自身输出结果，大幅降低人工监督成本。现已在 claude.ai、Claude Platform 及各主要云平台上线，Claude Code 同步新增 /ultrareview 命令和面向 Max 用户的扩展 auto 模式。

实践建议

  • 在 claude.ai 上切换到 Claude Opus 4.7，分配一项你此前觉得需要频繁干预的长链推理任务，观察其自动验证和精确指令跟随能力
  • 在 Claude Code 中使用新的 /ultrareview 命令，对一个含 3 个以上文件变更的 PR 做全量审查，与手动 review 对比覆盖率

2. Codex for (almost) everything

  • 来源: OpenAI Blog | 📡 也被 Sam Altman (Twitter), OpenAI (Twitter) 报道
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Codex computer use 开发者工具 插件 AI 编程

摘要: Codex 桌面客户端（macOS / Windows）迎来重大更新：新增计算机使用（computer use）能力，可并行操控 Mac 上所有应用而不干扰日常操作；内置浏览器支持网页搜索与交互；引入记忆功能，可从历史操作中学习并主动提出建议；同时集成 90+ 插件，涵盖文档、项目管理、代码审查、创意工作、部署等场景，并支持图像生成。

实践建议

  • 下载/更新 Codex 桌面客户端，开启 computer use 模式，让它在不干扰你手头工作的情况下并行处理一个日常开发任务（如运行测试套件）
  • 在插件市场挑选 2-3 个与你工作流相关的插件（如 Linear、GitHub、Notion），测试 Codex 跨工具协作的连贯性

3. Introducing GPT-Rosalind for life sciences research

  • 来源: OpenAI Blog | 📡 也被 OpenAI (Twitter) 报道
  • 分类: 模型发布
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: GPT-Rosalind 生命科学 药物发现 蛋白质推理 专业模型

摘要: OpenAI 推出 GPT-Rosalind 生命科学模型系列，这是一款面向药物发现、基因组分析、蛋白质推理和科学研究工作流优化的前沿推理模型。该模型从今天起以研究预览形式向 Amgen、Moderna、Allen Institute 及 Thermo Fisher Scientific 等合格客户开放，可通过 ChatGPT、Codex 和 API 使用。

实践建议

  • 如果你在生物医学领域，申请 GPT-Rosalind 研究预览访问权限，测试其在蛋白质功能预测或文献综述场景的表现
  • 通过 API 尝试 GPT-Rosalind 处理生物序列分析任务，与 GPT-5 通用推理模型进行对比评测

4. Qwen3.6-35B-A3B：现已开源

  • 来源: Qwen (Twitter)
  • 分类: 模型发布
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Qwen MoE 开源 多模态 智能体编程

摘要: Qwen 开源发布 Qwen3.6-35B-A3B，这是一款 MoE 架构模型，总参数 35B 仅需激活 3B，采用 Apache 2.0 许可证。该模型原生支持多模态（VLM + LM 双模式），在智能体编程任务上可媲美参数量 10 倍于自身的模型，视觉语言基准（RefCOCO 92.0、ODInW13 50.8）显著超越同规模竞品，空间智能尤为突出。

实践建议

  • 从 HuggingFace 下载 Qwen3.6-35B-A3B 并用 Ollama/LMStudio 本地部署，以编程任务（修复 bug、编写单元测试）对比与 Qwen3.5 同量级模型的质量差
  • 在 qwen.ai 或 Model Studio 体验其多模态推理能力，上传一张含文字和物体的复杂图片测试空间定位准确性

5. Accelerating the cyber defense ecosystem that protects us all

  • 来源: OpenAI Blog
  • 分类: 政策与安全
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: OpenAI 网络安全 GPT-5.4-Cyber Trusted Access 生态建设

摘要: 包括 CrowdStrike、Palo Alto Networks、SentinelOne 等领先安全公司和企业已加入 OpenAI 的 Trusted Access for Cyber 计划，可使用专为网络防御优化的 GPT-5.4-Cyber 模型。OpenAI 还设立 1000 万美元 API 授权金，帮助合格安全研究人员和防御方加速安全工具和能力的开发，进一步强化全球网络防御生态。

6. A new way to explore the web with AI Mode in Chrome

  • 来源: Google AI Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Chrome AI Mode 浏览器AI Google 网页交互

摘要: Google 对 Chrome 中的 AI Mode 进行重大升级，全面改变用户与网页的交互方式。新版 AI Mode 可在浏览过程中实时分析页面内容，提供上下文感知的摘要、问答和建议，帮助用户以对话方式探索和理解复杂网页内容，是 Google 将生成式 AI 深度嵌入浏览器体验的关键一步。

实践建议

  • 在 Chrome 中开启 AI Mode，用它浏览一篇长技术文章或研究报告，测试实时摘要与上下文问答的实用性

7. New ways to create personalized images in the Gemini app

  • 来源: Google AI Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐ (2/5)
  • 标签: Gemini 图像生成 个性化 Google Photos Nano Banana

摘要: Google Gemini 应用引入"Nano Banana 2"功能，利用用户的 Google Photos 个人照片库，在图像生成时融入个人上下文，创作出反映用户真实生活场景的个性化图像。这是 Google 在个人智能（Personal Intelligence）方向的一次具体落地，让 AI 生成内容与个人历史产生更深度的关联。

实践建议

  • 在 Gemini 应用中授权访问 Google Photos，生成一张融合你个人照片风格的形象图，评估个性化程度
