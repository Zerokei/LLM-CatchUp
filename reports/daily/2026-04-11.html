1. Introducing Claude Managed Agents

  • 来源: Anthropic Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐⭐ (5/5)
  • 标签: Anthropic Claude 托管智能体 AI Agent 企业AI

摘要: Anthropic 推出 Claude Managed Agents 公开测试版，提供托管在云端的 AI 智能体即服务，内置沙箱执行、检查点机制、凭证管理、权限管理和全链路追踪。Notion、Asana、Sentry 等企业已在此基础上构建智能体产品，官方宣称可将上线时间缩短 10 倍。这是继 Claude Code 之后 Anthropic 在智能体基础设施领域的又一重大战略落子。

实践建议

  • 在 Claude 平台注册，通过 Managed Agents API 部署一个简单的文件处理或数据分析智能体，体验内置沙箱和检查点机制
  • 对比 Managed Agents 与自建 LangChain/AutoGen 框架在延迟、可靠性和运维成本上的实际差异

2. OpenAI raises $122 billion to accelerate the next phase of AI

  • 分类: 商业动态
  • 重要性: ⭐⭐⭐⭐⭐ (5/5)
  • 标签: OpenAI 融资 估值 企业AI AI基础设施

摘要: OpenAI 完成 1220 亿美元融资，估值达 8520 亿美元，成为有史以来规模最大的科技初创公司融资轮次。亚马逊投入 500 亿美元、英伟达与软银各投入 300 亿美元；OpenAI 月收入突破 20 亿美元，企业业务占比超过 40%，API 每分钟处理超 150 亿个 token，Codex 周活用户突破 200 万。

3. Claude Cowork is now generally available

  • 来源: Anthropic Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Anthropic Claude Cowork 桌面应用 企业功能 智能体

摘要: Anthropic 将 Claude Cowork 从研究预览版升级为正式版，面向所有付费用户（macOS 和 Windows）开放。新增六项企业功能：基于角色的访问控制（RBAC）、小组消费限额、使用分析、扩展的 OpenTelemetry 支持、Zoom MCP 连接器以及工具级连接器权限控制，标志着 Anthropic 的自主计算机操作智能体正式进入企业市场。

实践建议

  • 升级到 Claude 任意付费计划后安装 Claude Desktop 应用，给 Cowork 一个具体目标（如整理本地文件夹、生成项目周报），观察其自主执行能力
  • 在企业环境中测试 RBAC 配置，确保不同团队只能调用经授权的 MCP 连接器

4. Anthropic expands partnership with Google and Broadcom

  • 来源: Anthropic Blog
  • 分类: 商业动态
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Anthropic Google Broadcom TPU 算力

摘要: Anthropic 宣布与 Google 和博通扩大算力合作，从 2027 年起通过博通定制芯片方案获得 3.5 吉瓦 Google TPU 算力（叠加现有 2026 年的 1 吉瓦）。Anthropic 营收年化运行率已突破 300 亿美元（较 2025 年底翻了三倍以上），此次大规模算力协议是应对需求爆炸式增长的战略应对。

6. OpenAI acquires TBPN

  • 来源: OpenAI Blog
  • 分类: 商业动态
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: OpenAI 收购 媒体 TBPN 品牌战略

摘要: OpenAI 以数亿美元收购科技谈话秀 TBPN（科技商业编程网络），成为其首次媒体公司并购。TBPN 是由创始人主持的日播直播节目，年营收超 3000 万美元，将纳入 OpenAI 战略部门并保持编辑独立性，此举旨在巩固 OpenAI 在科技创业者圈子中的话语权和公共叙事能力。
