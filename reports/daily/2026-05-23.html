1. Qwen3.7-Max 登陆 OpenRouter：第三方 Agent 测试压制 Opus 4.7 与 GPT-5.5

  • 来源: Qwen (Twitter)
  • 分类: 模型发布
  • 重要性: ⭐⭐⭐⭐⭐ (5/5)
  • 标签: Qwen3.7-Max OpenRouter Agent 基准 编码模型 Prompt 缓存

摘要: 阿里 Qwen 团队旗舰 Qwen3.7-Max 在 OpenRouter 上线，主打编码、办公生产力与长程自主执行；相较 Qwen3.6 在编码与 Agent 基准上显著提升，并支持显式 prompt 缓存。第三方 atomic_chat_hq 用「自训练 Tetris Bot」十轮 Agent 循环任务对比：Qwen3.7-Max 训练成本 $1.32、Bot 性能 +56%，分别比 Claude Opus 4.7（$12.15 / +28%）便宜 9 倍、比 GPT-5.5（$2.85 / +7%）便宜 2 倍，在长 Agent 循环这一维度宣告领先。

📎 多角度报道:

  • Qwen3.7-Max 上线 OpenRouter：聚焦编码与长程 Agent · Qwen (Twitter) · OpenRouter 上线公告

实践建议

  • 在 OpenRouter 接入 Qwen3.7-Max，于长 Agent 循环或编码任务上对比 Opus / GPT 成本
  • 对重复上下文的工作流启用显式 prompt 缓存以压低单次调用成本
  • 可借鉴 Tetris Bot 思路搭建自有 self-improving Agent 基准评估候选模型

2. Anthropic Glasswing 首月战报：Mythos Preview 挖出逾万高危漏洞

  • 来源: Anthropic Research
  • 分类: 研究
  • 重要性: ⭐⭐⭐⭐⭐ (5/5)
  • 标签: Project Glasswing Claude Mythos Preview 网络安全 漏洞披露 开源安全

摘要: Anthropic 披露 Project Glasswing 启动一个月战况：约 50 家关键基础设施合作伙伴使用 Claude Mythos Preview 共发现逾一万个高/严重级漏洞，Cloudflare 单家就上报 2,000 个 Bug，UK AISI、Mozilla、XBOW 等机构验证 Mythos Preview 表现领先。Anthropic 同步对 1,000+ 开源项目扫描，已确认 1,587 个真阳性漏洞，其中 530 个已披露、75 个已修复，并推出 Claude Security Public Beta 与威胁建模、扫描子代理等工具。文中坦言 Mythos 级模型尚未公开发布的原因——补丁部署速度已成新瓶颈，行业需缩短补丁周期、强化默认配置、推行 MFA。

📎 多角度报道:

  • Glasswing 项目一个月发现逾一万个关键软件漏洞 · Anthropic (Twitter) · 官方推特线程同步

3. Claude Code Auto 模式登陆 Pro 套餐，新增 Sonnet 4.6 支持

  • 来源: Claude Devs (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude Code Auto 模式 Sonnet 4.6 Opus 4.7 Pro 套餐

摘要: Claude Code 的 Auto 模式迎来两项更新：现已向 Pro 订阅用户开放，并在原有 Opus 4.7 之外新增 Sonnet 4.6 的支持。用户可通过 Shift+Tab 切换进入 Auto 模式，让 Claude 自动连续执行任务。 Anthropic 提示用户运行 claude update 或更新 Claude 桌面客户端至最新版本以使用 Auto 模式新功能，并给出权限模式文档链接说明如何通过 Auto 模式消除审批提示。

实践建议

  • Pro 订阅用户在 Claude Code 中按 Shift+Tab 启用 Auto 模式，将重复性自动化任务交给模型连贯执行
  • 对延迟敏感或预算受限的工作流切换至 Sonnet 4.6，保留 Opus 4.7 用于复杂推理场景

4. Project Genie 联手 Google Maps 街景生成可交互世界

  • 来源: Google DeepMind (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Project Genie Google Maps 街景 生成式世界 交互

摘要: Project Genie 正式与 Google Maps 街景集成，用户可将美国真实地点转化为新的、可交互的生成世界。这是世界模型与地理影像数据深度结合的首个公开产品形态，预示着「从真实地点起步的开放世界」可能成为生成式 3D 的主流入口。 Google DeepMind 将 Project Genie 的街景图像生成能力面向全球 Google AI Ultra 订阅者（18+）开放。用户可在 Google Labs 中点击地图图钉，选取美国任意地点和风格，将真实地理位置即时转化为可交互的生成世界。这是 Genie 世界模型从 Demo 走向规模化消费产品的关键一步。

实践建议

  • 关注 Genie+街景在生成时长、动作可控性上的实测表现，作为同类世界模型的对标基线
  • 针对地图/旅游/教育场景，构想以「真实坐标即提示」的产品玩法

8. Virgin Atlantic 用 Codex 按期发版：零 P1 缺陷、重构提速百倍

  • 来源: OpenAI Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Codex OpenAI 企业落地 代码重构 客户案例

摘要: 维珍航空借助 Codex 在圣诞旅行高峰前如期上线全新移动 App，达到接近完整的单元测试覆盖率并实现上线零 P1 缺陷。同时 Codex 让多年遗留代码的重构从数周缩短到数十分钟，代码量减少 78–80%；分析师团队也开始直接基于数据仓库自行构建内部工具，发布速度反而开始超出交付流程的承载能力。

实践建议

  • 在固定 deadline 的发版项目中，把 Codex 优先用于补齐单元测试覆盖率以降低 P1 风险
  • 针对维护多年的遗留模块，先用 Codex 做精简和重构试点，量化代码量与维护时长收益
  • 为非工程团队（数据分析、运营）开通 Codex 直接访问数据仓库的工作流，减少对中央数据团队的依赖

  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐ (3/5)
