2. Agent Token 成本节省被高估：Caveman 实测仅 8.5%

  • 来源: 宝玉的分享
  • 分类: 教程与观点
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Agent 成本 Token 优化 Caveman 上下文管理 工具调用

摘要: 文章复盘 Caveman 这类让 Agent 使用极简表达的 Skill，并引用 JetBrains 对 86 个真实编程任务的测试。结果显示，宣传中的 65% 输出 Token 节省在大样本中只有 8.5%，因为系统提示、代码、工具返回和上下文才是 Agent 账单的大头。更有效的成本优化应优先减少无关上下文、工具描述和返工。
