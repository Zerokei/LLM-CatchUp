### 2. [Claude Code 桌面端加入内置浏览器](https://x.com/ClaudeDevs/status/2075635283211772279)

- **来源**: Claude Devs (Twitter)
- **分类**: 产品与功能
- **重要性**: ⭐⭐⭐⭐ (4/5)
- **标签**: `Claude Code` `内置浏览器` `网页交互` `桌面应用` `开发智能体`

**摘要**: Claude Code 桌面版新增内置浏览器，Claude 可以打开文档、设计稿或其他网站，并像处理本地开发服务器一样阅读、点击和交互。该能力带有沙箱和可配置会话持久化选项，用户需要更新到最新版桌面应用并查阅外部站点浏览文档。对开发代理来说，这把查资料、看设计和验证网页状态纳入了同一个桌面工作流。 Claude Devs 提醒用户更新到最新版桌面应用，并指向关于浏览外部网站的文档。该条是同一线程的补充说明，重点是新桌面能力需要通过最新客户端获得，并可查阅配置与使用细节。

> **实践建议**
> - 升级 Claude Code 桌面版后，用项目文档、设计稿和本地开发服务器各跑一次端到端验证。
> - 为团队定义允许持久会话的外部站点清单，避免代理浏览器保留不必要的登录状态。
> - 把需要查文档和点击验证的开发任务改写成可复用的 Claude Code 流程。
