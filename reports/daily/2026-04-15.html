1. Claude Code 桌面端重构：多会话并行 + 集成终端与预览

  • 来源: Claude (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude Code 桌面端 多会话 开发者工具 IDE

摘要: Anthropic 重构了 Claude Code 桌面应用，引入侧边栏以便在同一窗口内并行运行多个 Claude 会话。新版本内置终端、文件编辑、HTML/PDF 预览与更快的 diff 查看器，支持拖拽式自定义布局；命令行插件可无缝运行。

实践建议

  • 从 claude.com/download 下载或更新 Claude 桌面应用，启动多个并行会话测试新侧边栏布局
  • 在同一窗口内分别使用不同会话处理代码修改与 PDF/HTML 预览，评估从 CLI 切换到桌面 IDE 风格工作流的收益

2. Claude Code 推出 Routines：可调度的长驻自动化任务

  • 来源: Claude (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude Code Routines 自动化 调度 Web 基础设施

摘要: Claude Code 以研究预览形式推出 Routines——一次配置 prompt、代码仓库与连接器即可按定时、API 调用或事件驱动反复运行。任务运行在 Anthropic 的 Web 基础设施上，无需保持本机开启；已适用于全部付费计划，CLI 中的 /schedule 自动迁移为 Routines。

实践建议

  • 创建一个 Routine：每天凌晨 2 点从 Linear 拉取最高优先级 bug，尝试修复并开 draft PR，观察一周后的命中率
  • 将团队已有的 /schedule CLI 作业检查一遍，确认迁移后仍按预期触发，同时接入事件驱动模式做对比

3. Google DeepMind 发布 Gemini Robotics-ER 1.6：更强空间推理，波士顿动力 Spot 自主读表

  • 来源: Google DeepMind (Twitter) | 📡 也被 Demis Hassabis (Twitter) 报道
  • 分类: 模型发布
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Gemini Robotics-ER 具身智能 Boston Dynamics 空间推理 机器人安全

摘要: Google DeepMind 推出 Gemini Robotics-ER 1.6，显著提升机器人对物理世界的视觉与空间理解，能独立规划并完成更复杂任务；与波士顿动力合作的 Spot 机器人已能自主读取复杂工业仪表。该版本也是迄今最安全的机器人模型，能识别液体、>20kg 重物等物理约束，视频中人类受伤风险检测较上代提升 10%。现已在 Google AI Studio 与 Gemini API 提供。

实践建议

  • 在 Google AI Studio 用现成的 Gemini Robotics-ER 1.6 端点上传一组机器人工作场景图片，测试空间定位与可执行步骤规划质量
  • 如果已有机器人/仿真环境，通过 Gemini API 替换现有 VLM 模块做 A/B 对比，重点评估对工业仪表、液体容器等边界场景的识别率

4. Anthropic 发布 Automated Alignment Researcher：让 Claude Opus 4.6 加速可扩展监督研究

  • 来源: Anthropic Research | 📡 也被 Anthropic (Twitter) 报道
  • 分类: 研究
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: 对齐研究 可扩展监督 Claude Opus 4.6 Anthropic Fellows 自动化科研

摘要: Anthropic Fellows 项目发布研究成果：探索让 Claude Opus 4.6 自动化开展一个关键对齐问题——用较弱模型监督更强模型训练（scalable oversight）。实验表明 Claude 可显著提升实验与探索速率，但在缺少可验证指标的"模糊"对齐研究上仍力不从心，AI 尚未成为通用对齐科学家。研究展示了用 AI 加速对齐科研的可行路径，同时明确当前局限。

6. Chrome 推出 Skills：把常用 AI prompt 一键化并可复用

  • 来源: Google AI Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: Chrome Gemini Skills 提示工程 浏览器 AI

摘要: Google 在 Chrome 中推出 Skills 功能，用户可以发现、保存并 remix 他人的 AI 工作流，把高质量 prompt 固化为一键触发的工具并在浏览器中随时复用。该功能瞄准重复性的 AI 使用场景，降低 prompt 工程门槛。

实践建议

  • 在 Chrome 中开启 Skills，保存 2-3 个你最常用的日常 AI prompt（翻译、摘要、代码解释），比较一键调用与手动粘贴的效率差
  • 浏览社区 Skills 目录，fork 一个生产力类 skill 再做局部改动，观察 remix 机制的可维护性
