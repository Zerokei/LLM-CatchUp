### 1. [Anthropic 发布 Claude Sonnet 5](https://www.anthropic.com/news/claude-sonnet-5)

- **来源**: Anthropic Blog
- **分类**: 模型发布
- **重要性**: ⭐⭐⭐⭐⭐ (5/5)
- **标签**: `Claude Sonnet 5` `模型发布` `智能体` `Claude Code` `编码能力`

**摘要**: Anthropic 发布 Claude Sonnet 5，定位为更强的 Sonnet 级代理模型，能规划、使用浏览器和终端等工具，并在推理、工具调用、编码和知识工作上显著优于 Sonnet 4.6。它已面向所有计划开放，并进入 Claude Code 与 Claude Platform；在 BrowseComp、OSWorld-Verified 等代理评测中接近 Opus 4.8，同时价格更低。

> **实践建议**
> - 在现有 Claude Code 工作流中用 `claude-sonnet-5` 复测多步骤修复、重构和浏览器工具任务。
> - 按不同 effort 档位跑一组内部基准，比较 Sonnet 5 与 Opus 4.8 的成本和成功率。
> - 更新模型路由策略，把中高复杂度代理任务优先分流到 Sonnet 5。
