1. Anthropic 推出面向美国 K-12 教师的 Claude

  • 来源: Anthropic Blog
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude 教育科技 K-12 Learning Commons 隐私

摘要: Anthropic 发布 Claude for Teachers，为美国已验证的 K-12 教育工作者免费提供高级 Claude 能力、教学技能库，以及与 50 州学术标准映射的 Learning Commons 连接。该产品可生成标准对齐的课程计划、分层教学材料、自动评分练习，并通过 ASSISTments、Brisk Teaching、Canva Education、MagicSchool 等教育工具扩展课堂工作流，同时声明教师数据不用于模型训练并覆盖 FERPA 相关隐私要求。

实践建议

  • 教育团队可用本州课程标准测试课程计划生成、分层材料改写和学生数据分析三类核心工作流。
  • 在试点前梳理学生数据共享边界，确认 FERPA、校区政策和工具连接权限。

2. Anthropic 聚焦加拿大：Claude 使用研究 + 研究投入

  • 来源: Anthropic Research
  • 分类: 研究
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude 加拿大 经济指数 AI 采用 研究资助

摘要: Anthropic 发布加拿大 Claude 使用研究，并同时承诺投入 1000 万加元支持当地 AI 研究生态。经济指数显示，加拿大用户在软件开发、教育、写作和业务分析中广泛使用 Claude，且使用模式可为地区劳动力影响和技能迁移提供量化线索。配套研究资助把这份区域观察上升为长期生态建设，重点落在安全、治理和本地研究能力。

3. Claude Artifacts 升级：公开分享 + 多人编辑 + Slack 生成

  • 来源: Claude Devs (Twitter)
  • 分类: 产品与功能
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude Artifacts Claude Code 多人协作 Slack 内部工具

摘要: Claude Artifacts 获得一组协作升级：公开分享、多用户共同编辑，以及从 Slack 线程生成内部 Artifact。更新把 Artifacts 从个人产物推进到团队协作界面，适合把讨论、仪表盘和轻量应用快速转成可共享工作件。第三方评价也强调，这些能力可以和现有 Claude 工作流组合出更灵活的内部工具。

📎 多角度报道:

  • Claude Artifacts 面向团队推出多人编辑 · Claude Devs (Twitter) · 多人编辑演示
  • Claude Artifacts 协作能力带来更灵活工作流 · Thariq (Twitter) · 组合工作流点评

实践建议

  • 用一个真实 Slack 讨论生成内部仪表盘，验证数据结构和权限边界。
  • 把需要多人校对的 Artifact 改成协作编辑流程，记录冲突和版本管理问题。
  • 测试公开分享链接是否满足团队对访问控制和外部展示的要求。

4. Anthropic 研究 Claude 价值表达的模型与语言差异

  • 来源: Anthropic (Twitter)
  • 分类: 研究
  • 重要性: ⭐⭐⭐⭐ (4/5)
  • 标签: Claude 价值对齐 模型行为 多语言 安全评估

摘要: Anthropic 延续其价值观测研究，比较 Claude 在不同模型和语言环境下表达价值的方式。研究发现，价值表达会随模型版本和对话语言变化，在温暖与严谨、服从与谨慎等轴线上呈现可测差异。团队强调这类监测有助于判断模型行为是否符合预期，并为未来训练和部署提供更细粒度的反馈。

📎 多角度报道:

  • Claude 的价值表达随语言显著变化 · Anthropic (Twitter) · 语言差异结果

7. xAI 说明 Grok Build 隐私控制

  • 来源: xAI (Twitter)
  • 分类: 政策与安全
  • 重要性: ⭐⭐⭐ (3/5)
  • 标签: xAI Grok Build 隐私 零数据保留 CLI

摘要: xAI 表示，启用零数据保留的团队不会保留 trace 和代码数据，使用 API key 的 Grok Build 也遵守 ZDR 设置。若未启用 ZDR，CLI 中的「/privacy」命令可关闭数据保留并删除此前同步的数据，用户也可随时查看或修改相关设置。

8. DeepMind 展示 Antigravity 历史预测技能

  • 重要性: ⭐⭐⭐ (3/5)
