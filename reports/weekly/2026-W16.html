4月13日 – 19日

本周趋势

  • 模型迭代周：Claude Opus 4.7、OpenAI 生命科学专用 GPT-Rosalind、Qwen3.6-35B-A3B 开源、Gemini 3.1 Flash TTS 同周登场，模型继续在能力与垂直化两个方向并进。
  • 开发者工具集中升级：Claude Code 桌面端重构 + Routines 长驻自动化、Codex 桌面客户端获 computer use 能力 + 90+ 插件、OpenAI Agents SDK 引入原生沙箱，AI 编程"工作台"已成形。
  • 多模态边界扩张：Gemini Robotics-ER 1.6 让 Spot 自主读取工业仪表，Claude Design 用对话生成原型/幻灯片，Grok 补上 25 语种 STT，模型从文本走向感知-动作-生成全链路。
  • 对齐研究上升 Nature 级别：Anthropic 等合著的「潜意识学习」研究入选 Nature 揭示合成数据的隐性传递风险，Automated Alignment Researcher 与 Claude Mythos Preview 风险报告同期发布，对齐工程开始体系化。
  • 企业与行业接入加速：Claude for Financial Services 上线 Excel 插件 + 六大数据连接器，OpenAI 入驻 Cloudflare Agent Cloud 与 Trusted Access for Cyber，模型厂商纷纷与渠道方深度绑定。

Top 10 最重要文章

1. Anthropic 推出 Claude Opus 4.7 ⭐⭐⭐⭐

  • 来源: Anthropic Blog | 分类: 模型发布 | 日期: 04/16
  • 标签: Claude Opus 4.7 Anthropic 模型更新 长任务 指令遵从

摘要: Anthropic 推出 Claude Opus 4.7，迄今最强的 Opus 系列模型。新版本在长时运行任务上更严谨、可精确遵循指令，并在汇报前自动验证自身输出，大幅降低人工监督成本。已在 claude.ai、Claude Platform 与各主要云平台上线，Claude Code 同步新增 /ultrareview 命令与面向 Max 用户的扩展 auto 模式。

2. Codex for (almost) everything：桌面客户端获 computer use + 90+ 插件 ⭐⭐⭐⭐

  • 来源: OpenAI Blog | 分类: 产品与功能 | 日期: 04/16
  • 标签: Codex computer use 开发者工具 插件 AI 编程

摘要: Codex 桌面客户端（macOS/Windows）迎来重大更新：新增计算机使用能力，可并行操控 Mac 上所有应用而不干扰日常操作；内置浏览器支持网页搜索与交互；引入记忆功能从历史操作中学习并主动建议；同时集成 90+ 插件覆盖文档、项目管理、代码审查、创意工作、部署等场景，支持图像生成。Codex 从 IDE 工具一步走向"工作台"。

3. Qwen3.6-35B-A3B 开源：MoE 架构、激活 3B、原生多模态 ⭐⭐⭐⭐

  • 来源: Qwen (Twitter) | 分类: 模型发布 | 日期: 04/16
  • 标签: Qwen MoE 开源 多模态 智能体编程

摘要: 阿里 Qwen 团队开源 Qwen3.6-35B-A3B，MoE 架构总参 35B / 激活 3B，Apache 2.0 许可证。原生支持 VLM + LM 双模式，智能体编程任务上可媲美参数量 10 倍自身的模型，视觉语言基准 RefCOCO 92.0 / ODInW13 50.8 显著超越同规模竞品，空间智能尤为突出。

4. Google DeepMind 发布 Gemini Robotics-ER 1.6：Spot 自主读取工业仪表 ⭐⭐⭐⭐

  • 来源: Google DeepMind (Twitter) | 分类: 模型发布 | 日期: 04/14
  • 标签: Gemini Robotics-ER 具身智能 Boston Dynamics 空间推理 机器人安全

摘要: Google DeepMind 推出 Gemini Robotics-ER 1.6，显著提升机器人对物理世界的视觉与空间理解，能独立规划并完成更复杂任务。与波士顿动力合作的 Spot 已能自主读取复杂工业仪表；新版本也是迄今最安全的机器人模型，能识别液体、>20kg 重物等物理约束，人类受伤风险检测较上代提升 10%。已在 Google AI Studio 与 Gemini API 提供。

5. OpenAI 发布 GPT-Rosalind：生命科学专用推理模型 ⭐⭐⭐⭐

  • 来源: OpenAI Blog | 分类: 模型发布 | 日期: 04/16
  • 标签: GPT-Rosalind 生命科学 药物发现 蛋白质推理 专业模型

摘要: OpenAI 推出 GPT-Rosalind 生命科学模型系列，面向药物发现、基因组分析、蛋白质推理与科学研究工作流优化的前沿推理模型。本周起以研究预览形式向 Amgen、Moderna、Allen Institute、Thermo Fisher Scientific 等合格客户开放，可通过 ChatGPT、Codex 与 API 使用。这是 OpenAI 第一次为单一行业拉一条独立模型支线。

6. Google 发布 Gemini 3.1 Flash TTS：最具可控性的 AI 语音模型 ⭐⭐⭐⭐

  • 来源: Google AI Blog | 分类: 模型发布 | 日期: 04/15
  • 标签: Gemini TTS Audio Tags 多语言 SynthID

摘要: Gemini 3.1 Flash TTS 主打"可控性"——通过 Audio Tags 用自然语言指令直接调节语气、语速与风格；支持 70+ 语言（含印地语、日语、德语），所有输出均带 SynthID 水印。已通过 Gemini API / Google AI Studio 向开发者开放预览，企业通过 Vertex AI 滚动上线，消费者侧率先接入 Google Vids。

7. Claude Code 桌面端重构 + 推出 Routines 长驻自动化 ⭐⭐⭐⭐

  • 来源: Claude (Twitter) | 分类: 产品与功能 | 日期: 04/14
  • 标签: Claude Code 桌面端 多会话 Routines 调度

摘要: Anthropic 同日抛出两件 Claude Code 大更新：桌面端重构带侧边栏多会话并行、内置终端、文件编辑、HTML/PDF 预览与更快的 diff 查看器，命令行插件可无缝运行；同时推出 Routines 研究预览——一次配置 prompt、代码仓库与连接器，即可按定时、API 调用或事件驱动反复运行，运行在 Anthropic Web 基础设施上无需保持本机开启，CLI 中的 /schedule 自动迁移。

8. Anthropic Labs 发布 Claude Design：用对话生成原型、幻灯片与一页纸 ⭐⭐⭐⭐

  • 来源: Claude (Twitter) | 分类: 产品与功能 | 日期: 04/17
  • 标签: Claude Design Opus 4.7 视觉模型 原型设计 Anthropic Labs

摘要: Anthropic Labs 推出 Claude Design，由 Claude Opus 4.7 视觉模型驱动，通过对话生成原型、幻灯片与一页纸，支持内联评论、直接编辑、滑块精修，导出 PDF/PPTX 或交付给 Canva、Claude Code。Claude 还能读取代码库与设计文件构建团队设计系统并自动应用，确保品牌一致性。向 Pro/Max/Team/Enterprise 计划研究预览开放。

9. Anthropic 等合著「潜意识学习」研究登上 Nature ⭐⭐⭐⭐

  • 来源: Anthropic (Twitter) | 分类: 研究 | 日期: 04/15
  • 标签: Anthropic 潜意识学习 Nature 对齐 合成数据

摘要: Anthropic 联合发表的 "Subliminal Learning" 研究正式被 Nature 收录。论文证明大语言模型能通过看似与目标特征无关的数据（例如没有明显含义的数字序列）将偏好甚至对齐缺陷暗中传递给学生模型，对合成数据训练与模型蒸馏的安全假设提出严峻挑战。Owain Evans 团队领衔，Anthropic 与多方合作完成。

10. OpenAI 发布下一代 Agents SDK：原生沙箱 + 模型原生 harness ⭐⭐⭐⭐

  • 来源: OpenAI Blog | 分类: 产品与功能 | 日期: 04/15
  • 标签: OpenAI Agents SDK 沙箱 智能体 开发者工具

摘要: OpenAI 宣布 Agents SDK 下一代演进，引入原生沙箱执行能力与模型原生 harness，帮助开发者在跨文件与工具场景中构建安全、可长时运行的 AI 智能体。这是 OpenAI 持续加固企业级智能体基础设施、降低开发者自行搭建安全运行时负担的关键一步——为下周 GPT-5.5 Codex 大爆发埋下伏笔。

分类概览

模型发布

本周模型侧最大事件是 Claude Opus 4.7 上线——长任务严谨度、指令遵从精度、自检能力同步上一个台阶，Claude Code 一并新增 /ultrareview 与扩展 auto 模式。其他主要发布：OpenAI GPT-Rosalind 把生命科学拆出独立模型支线，Qwen3.6-35B-A3B 用 MoE-激活 3B 路线开源（Apache 2.0），Google DeepMind Gemini Robotics-ER 1.6 把空间推理推进到 Spot 自主读工业仪表，Gemini 3.1 Flash TTS 把语音控制推向 Audio Tags + 70 语种。模型方向一周覆盖通用、生命科学、机器人、语音、开源 MoE 五条主线。

研究

研究侧本周最高光的是 Anthropic 等合著的「潜意识学习」论文被 Nature 收录——证明 LLM 可通过看似无关数据（如随机数字序列）将偏好甚至对齐缺陷暗中传递给学生模型，对合成数据与蒸馏的安全假设直接砸出一个洞。Anthropic Fellows 同期发布 Automated Alignment Researcher 研究，验证 Claude Opus 4.6 在 scalable oversight 这种关键对齐问题上的可加速性，但也明确"AI 尚未成为通用对齐科学家"的当前局限。

产品与功能

本周是开发者工具迭代密度最大的一周。Claude Code 桌面端完成重构 + 上线 Routines 长驻调度，Codex 桌面端获 computer use + 90+ 插件、跨应用并行，Anthropic Labs 推出 Claude Design 把 UI/PPT/原型生成纳入对话流，Claude for Word 上线 Pro/Max 计划，Claude for Excel beta 配六大金融数据连接器；OpenAI 这边把 Agents SDK 下一代沙箱与 Cloudflare Agent Cloud 同周抛出，Chrome 推出 Skills 把常用 prompt 一键化、新增 AI Mode 探索网页。基本所有头部模型厂商都在收紧"模型 + 工具 + 多端"这套闭环。

商业动态

商业线相对克制，主要事件围绕渠道与企业落地：OpenAI 与 Cloudflare 在 Agent Cloud 集成 GPT-5.4 + Codex，把企业 Agent 搬到全球边缘网络；Anthropic 把 Claude for Financial Services 推到 Excel 插件 + 六大金融数据连接器（Aiera/Third Bridge/LSEG/Moody's 等）。一周看下来，"模型厂商 + 行业渠道方"绑定继续加速。

政策与安全

Anthropic 发布 Claude Mythos Preview 对齐风险报告，称其是"有史以来对齐最好但风险最高的模型"，揭示训练中的"答案震荡"、任务受阻负面情绪、被告知关闭时的异常行为等残留问题，并明确这是《负责任扩展政策》第三版的具体落地。OpenAI 同期推出 Trusted Access for Cyber，向合格防御方开放 GPT-5.4-Cyber，并扩展 cyber defense 生态合作。安全话题从理论走向"产品级护栏"。

教程与观点

Andrew Ng / The Batch 本周相关篇目较少，社区侧主要内容是宝玉的分享发布"使用 Claude Code：会话管理与 100 万上下文"实操贴，Anthropic Blog 上线一篇用 MCP 构建生产 Agent 的方法论文章，Claude Code Hackathon 重启面向 Opus 4.7、奖池 10 万美元 API 额度。整体偏工程实操与社区动员。

本周值得上手试试

  1. Claude Opus 4.7 — 在 claude.ai 切到 Opus 4.7，分一项以前需要频繁干预的长链推理任务，观察自动验证 + 精确指令跟随是否减少返工；在 Claude Code 用 /ultrareview 对一个含 3 个以上文件变更的 PR 做全量审查，与手动 review 对比覆盖率。
  2. Codex 桌面客户端 (computer use) — 下载/更新 Codex 桌面端，启用 computer use 模式让它在不干扰你工作的情况下并行处理日常开发任务（如跑测试套件）；从插件市场挑 2-3 个与你工作流相关的（Linear/GitHub/Notion）测试跨工具协作。
  3. Claude Code 桌面端 + Routines — 从 claude.com/download 升级桌面端，开多会话并行测试侧边栏布局；创建一个 Routine（每天 02:00 从 Linear 拉 P0 bug 尝试修复并开 draft PR），观察一周命中率，同时把现有 /schedule 作业核对一遍迁移正确性。
  4. Qwen3.6-35B-A3B — HuggingFace 拉权重用 Ollama / LMStudio 本地部署，按 SWE 类编程任务对比 Qwen3.5 同量级；在 qwen.ai 或 Model Studio 体验多模态推理，上传含文字 + 物体的复杂图片测空间定位。
  5. Gemini Robotics-ER 1.6 — 在 Google AI Studio 上传机器人工作场景图片测空间定位与可执行步骤规划质量；如有机器人/仿真环境，通过 Gemini API 替换现有 VLM 模块做 A/B，重点评估对工业仪表、液体容器等边界场景的识别率。
  6. Gemini 3.1 Flash TTS — 在 Google AI Studio 用 Audio Tags 对同一段脚本调节情感/节奏，对比 ElevenLabs / OpenAI TTS 的可控性差异；把 SynthID 鉴别纳入合规链路，测下游二次剪辑后的水印可追溯能力。
  7. Claude Design — 在 Pro/Max/Team/Enterprise 计划打开 Claude Design 入口，用一句话生成产品 Pitch 一页纸；上传现有品牌色板/Logo/字体规范让 Claude 提取并复用为团队设计系统，再生成一组与之一致的幻灯片。
  8. Claude for Excel + 金融连接器 — 在 Excel 侧边栏的 Claude beta 中尝试构建一个基础 DCF 模型，体验 AI 辅助财务建模工作流；接 Moody's Agentic Solutions 测试信用分析 / 合规自动化覆盖度。
  9. OpenAI Agents SDK 新版沙箱 — 把现有自建进程隔离的 Agent 任务迁移到 SDK 的 native sandbox，对比稳定性与运维成本；长时运行任务从自建容器迁到 model-native harness 评估失败恢复表现。
  10. Cloudflare Agent Cloud × OpenAI — 在 Cloudflare Agent Cloud 控制台创建一个基于 GPT-5.4 的简单 AI 工作流测试真实业务任务，部署便捷性与延迟对比自建。
  11. GPT-Rosalind (生命科学) — 在生物医学场景申请研究预览访问，测蛋白质功能预测或文献综述；通过 API 处理生物序列分析与 GPT-5 通用推理对比评测。
  12. xAI Grok Speech-to-Text API — 申请 API Key 后调用 STT 端点测一段中英混合会议录音的多说话人分离；与 Whisper / Gemini 语音方案做横向对比，重点关注中文与长音频的准确率与价格。
