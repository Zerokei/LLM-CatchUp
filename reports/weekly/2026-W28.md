# CatchUp 周报 — 2026-W28 (07/06 - 07/12)

## 本周趋势

- **OpenAI模型换代**：GPT-5.6 Sol、Terra、Luna 与 GPT-Live 同周上线，把文本、办公、编码和语音入口一起更新。
- **代理产品跨端化**：ChatGPT Work、Claude Cowork 与 Claude Code 内置浏览器都在把长任务代理从单点工具推向跨设备工作流。
- **语音交互升温**：GPT-Live、GPT-Realtime-2.1-mini 和 Grok 新语音显示实时语音代理正在追求更低延迟和更自然打断。
- **媒体模型转代理**：Meta Muse、Muse Spark 1.1 与字节 Seedream 5.0 Pro 将图像视频生成推进到工具调用、精修和专业设计流程。
- **可解释性更实证**：Anthropic J-space、Google DeepMind 可解释性讨论与双用途知识模块研究让模型审计继续贴近内部机制。
- **企业采用更具体**：AP+、MUFG、德国电信与阿尔伯塔政府案例显示 AI 已进入支付、银行、电信和公共部门的实际流程。

## 分类概览

### 模型发布

本周模型发布由 OpenAI 的 GPT-5.6 系列牵引，并延伸到实时语音、全双工语音和低成本模型扩散。Sol、Terra、Luna 的三档定位让模型选型更像产品组合管理，而不是单一旗舰替换。Meta Muse、Muse Spark 1.1 与字节 Seedream 5.0 Pro 则把多模态模型竞争带向专业媒体生成、设计编辑和 API 分发。整体看，模型发布正在同时服务开发者、企业办公、语音代理和创意生产多个入口。

### 研究

研究线索集中在两个方向：理解模型内部，以及衡量代理在真实环境中的学习能力。Anthropic 的 J-space 研究把 Claude 的内部工作区变成可读、可干预和可演示的对象，为安全审计提供了更细的抓手。ByteDance Seed 的 EdgeBench 用长时真实任务和缩放律描述环境学习，为代理评测补上了时间维度。Google DeepMind 的机器人数据与可解释性讨论也提醒行业，下一阶段评测会越来越依赖真实环境、可复验数据和机制层证据。

### 产品与功能

产品与功能的主线是代理工作流从工具扩展到工作空间。ChatGPT Work 把 Codex 能力嵌入跨应用办公长任务，Claude Cowork 则向移动端、网页端和统一主页扩张，Claude Code 内置浏览器把外部资料、设计稿和网页验证纳入桌面开发流程。Google Gemini API Managed Agents、OpenAI Sites、Build Week 和 Claude Managed Agents 共同说明，平台方正在把长任务、工具连接、原型发布和成本路由做成托管能力。对团队而言，本周最值得关注的是如何设置权限、审计、验收和成本边界。

### 商业动态

商业动态数量明显上升，重点不是融资或估值，而是企业与公共部门的实际采用案例。AP+、MUFG 和德国电信都把 ChatGPT、Codex 或 OpenAI API 放进受监管行业的真实流程，强调初稿、调查、原型和运营优化的效率收益。阿尔伯塔政府用 Claude 扫描大规模遗留代码，显示公共部门也在把代理用于安全和现代化工程。Anthropic 长期利益信托引入伯南克，则把 AI 的宏观经济影响和公司治理继续绑定。

### 政策与安全

政策与安全本周围绕生物安全、双用途知识和政府网络安全实践展开。OpenAI 将 Bio Bounty 转为持续私有计划，并把通用越狱奖励提高到 5 万美元，说明前沿模型发布后的红队测试会持续常态化。Anthropic 的 GRAM 研究尝试把双用途知识路由到可移除模块中，为未来模型能力管控提供了不同于事后遗忘的路径。阿尔伯塔政府案例则从防护侧说明，大规模代码审计已经成为政府使用 AI 的现实场景。

### 教程与观点

教程与观点部分最有价值的是对代理式开发边界的梳理。宝玉的分享把 Claude Code 循环、ChatGPT Work 与 Codex 的任务分工讲清楚，Thariq 的内容则强调代理式编程需要不断缩小未知，而不是直接把模糊任务丢给模型。OpenAI Build Week 和 Codex 生物工具访谈把这些方法拉回真实项目实践。DeepMind 关于思维链与可解释性的播客，也为理解模型审计和安全监控提供了更基础的讨论入口。

## 本周值得上手试试

1. **GPT-5.6 Sol/Terra/Luna** — 用现有代码、办公文档、长任务和成本基准分别测试三档模型，记录质量、延迟和每美元任务完成率。
2. **GPT-Live 与 Realtime API** — 选择客服、会议助手或语言练习场景，重点验证打断、背景噪声、工具调用和 p95 延迟。
3. **ChatGPT Work** — 挑一个跨文档、表格和网页的真实项目，设置文件权限、审计日志和人工验收点后试跑长任务交付。
4. **Claude Cowork** — 把异步研究、代码整理或文档任务迁入 Cowork，观察桌面到移动端的交接、通知和 Artifacts 完整性。
5. **Claude Managed Agents** — 把昂贵模型限定为顾问或编排器，让低价模型处理长上下文执行，并比较全强模型方案的成本质量比。
6. **Meta Muse 与 Seedream 5.0 Pro** — 用真实品牌素材、信息图、中文海报和局部编辑任务测试文字、坐标和多参考图稳定性。
7. **Gemini API Managed Agents** — 盘点现有长时间运行代理任务，评估后台任务和远程 MCP 是否能减少自建运行基础设施。
8. **Claude Code 内置浏览器** — 用外部文档、设计稿和本地开发服务器跑一次端到端开发验证，并定义可持久会话站点清单。
9. **OpenAI Sites** — 选一个内部小工具快速生成可分享站点，记录发布、权限、地区和配额限制。
10. **Claude Code 循环** — 为高频开发任务写入明确停止条件、测试命令和 SKILL.md，把一次性提示改成可重复例行程序。
