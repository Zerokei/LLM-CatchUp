## 分类概览

### 模型发布

模型发布本周由两条主线主导：Google I/O 2026 把 Gemini 3.5 Flash 与世界模型 Gemini Omni 同台推出，3.5 Flash 在编码与 Agent 基准上反超上一代 Pro 并配套 Antigravity 2.0、第 8 代 TPU，3.5 Pro 锁定下月发布；阿里 Qwen3.7-Max 则以 Artificial Analysis 56.6 分、100 万 token 上下文与显式 Prompt 缓存进入旗舰梯队，并在 LMSys Arena 文本榜跻身第 6、视觉榜第 5 实验室。第三方 Tetris Bot 长程 Agent 任务测得 Qwen3.7-Max 单次训练成本 $1.32，较 Claude Opus 4.7 便宜 9 倍，成为本周最具说服力的性价比对照。Claude Code Fast 模式默认切到 Opus 4.7、Sonnet 4.6 进入 Auto 模式，进一步把前沿模型铺到日常编码工作流中。

### 研究

研究面最重的两条：OpenAI 通用推理模型独立攻克 Erdős 1946 年平面单位距离猜想，推翻 80 年「方格最优」直觉，证明意外用上代数数论的类域塔与 Golod–Shafarevich 理论，被 Tim Gowers 称为「AI 数学的里程碑」；Anthropic Project Glasswing 一个月内借助 Claude Mythos Preview 在关键基础设施和开源生态中挖出逾一万个高 / 严重漏洞，把漏洞挖掘从「找得到」推进到「来不及修」的新形态。两条都强调通用推理模型对真实科学与安全问题的端到端解决能力。Google DeepMind 同步推出 Gemini for Science 与基于 AlphaEvolve 的计算发现原型，把多代理「想法竞赛」机制带进流行病学等领域。

### 产品与功能

产品侧密度最高，主轴是「长程自主 + 多设备协同」：Codex 推出 Appshots 窗口注入、Mac 锁屏远程操控、跨设备续接 ChatGPT 手机端，大仓库 Git 操作提速 10–50 倍；Claude Code Auto 模式下放 Pro 套餐、新增 Sonnet 4.6，Console 端上线 Prompt 缓存诊断面板。安全部署侧，Claude Managed Agents 同时端出自托管沙箱与 MCP 隧道，回应企业「Agent 进生产」的数据边界焦虑。Google 把 Antigravity 升级到 2.0 任务中控并附 CLI、SDK、Science Skills，Project Genie 联手 Google Maps 街景将真实地理位置转成可交互生成世界。生态边缘，xAI 在 opencode / openclaw 把 Grok 与 X Premium 订阅打通，Anthropic 同步发布 Computer Use 生产实践指南与大型代码库落地实践，整体节奏明显从「能跑」转向「能稳定上线」。

### 商业动态

商业层面是一轮典型的「版图重组周」：Anthropic 收购 SDK / MCP 工具平台 Stainless，与 KPMG 达成全球联盟把 Claude 嵌入 Digital Gateway 向 27.6 万员工铺开，且成为其私募股权领域首选合作伙伴；同时 Andrej Karpathy 宣布加入 Anthropic，人才信号叠加渠道与生态。OpenAI 一边向当批 YC 创业公司每家投入 200 万美元 token 额度换股权，一边推出 Guaranteed Capacity 让客户以 1–3 年承诺锁定算力，并与 Dell 合作把 Codex 带进本地与混合云；当选 2026 Gartner 企业 AI 编码 Agent 领导者象限。Google 在密苏里州追加 500MW 数据中心容量与 2000 万美元能源基金，DeepSeek 把 V4-Pro API 折扣永久化以巩固开源云服务定价预期。

### 政策与安全

政策与安全本周相对集中：内容溯源侧，OpenAI 升级为 C2PA 合规生成产品，并与 Google 合作把 SynthID 不可见水印加进 ChatGPT / Codex / API 生成的图像，附公开验证工具；Google DeepMind 同步把 SynthID 扩展到更多合作伙伴生成的 AI 内容，并在 Gemini App 与 Search 提供「这段内容是否 AI 生成」查询入口。Anthropic 一边以 Project Glasswing 把 AI 漏洞挖掘规模化用于关键基础设施加固，一边公布与神经科学、宗教与跨文化伦理学者的「道德养成」对谈：把 Claude 的伦理承诺改造成可在任务中主动调用的工具，已显著降低多项内部对齐评测中的失准率。透明度、可追溯与可控行为正在被同步推进。

### 教程与观点

教程与观点延续两条偏实践的脉络：一是「让代理在长程任务里自己留笔记」，Thariq 推出 implementation-notes 提示词模板，要求 Agent 在实现规格时同步记录设计决策、偏离与开放问题，并把「HTML 是新的 Markdown」展开为可交互规格与一次性微 UI 的工作流；二是企业落地经验集中产出，OpenAI 公布 Ramp 用 Codex + GPT-5.5 把代码评审从数小时压到数分钟、Virgin Atlantic 借 Codex 在圣诞高峰前如期上线零 P1 缺陷的移动 App、AdventHealth 把 ChatGPT 当「产品」铺开并以采用率作 KPI。Anthropic 同步发布 Computer Use 生产实践指南、Claude Code 大型代码库落地实践与「The Problem Solvers」创业者系列，整体把焦点从「模型够不够强」转向「怎么让代理稳定上线、可被管理」。Sam Altman 把 OpenAI 下一阶段重点抛给「个人 AGI」，是个值得标记的方向信号。

## 本周值得上手试试

1. **Gemini 3.5 Flash** — 在 Google AI Studio 取 Gemini API key 跑通最小调用，然后在自家延迟 / 成本敏感的智能体工作流中以混合调度对比当前前沿模型；为下月 Gemini 3.5 Pro 预留模型配置参数便于快速接入评测。
2. **Gemini Omni / Google Flow** — 用 Omni Flash 上传一段自拍视频，用自然语言改写动作或场景，验证多模态编辑链路；同时关注 Omni API 开放窗口以规划内容生成集成。
3. **Antigravity 2.0** — 在 2.0 任务中控里把一个真实项目拆给多个子代理并行处理，并通过 Antigravity CLI / SDK 把现有工程脚本接入；如做生命科学相关研究可启用 Science Skills 接 UniProt、AlphaFold 等 30+ 数据源。
4. **Qwen3.7-Max** — 通过 OpenRouter 或阿里云百炼接入 Qwen3.7-Max，在编码与长 Agent 循环任务上跑一遍自家工作流，对重复上下文启用显式 Prompt 缓存压低单次成本，并搭一组自训练「Tetris Bot」式自有基准做横向评估。
5. **Claude Code Auto / Fast 模式** — `claude update` 升级后 Pro 套餐 Shift+Tab 进入 Auto 模式让 Sonnet 4.6 连贯执行重复性任务；时间敏感场景执行 `/fast` 切到 Opus 4.7 的 Fast 模式，正式启用前先看 `code.claude.com/docs/en/fast-mode` 评估按 token 更高费率的成本。
6. **Claude Managed Agents 自托管沙箱与 MCP 隧道** — 把 Agent 执行环境放在自有基础设施 / 托管沙箱上以满足数据驻留与合规；用 MCP 隧道安全暴露内网服务给 Agent，无需对公网开放；长会话中可热替换工具或 MCP 配置而无需重启。
7. **Anthropic Computer Use 生产实践** — 按任务难度分级设置 thinking effort 避免过度推理，对长会话主动做上下文压缩或分段，并录制关键操作演示供 Claude 回放以提升 UI 点击准确率。
8. **Claude Console Prompt 缓存诊断** — 登录 `platform.claude.com/usage/cache` 回看近期未命中请求，把易变化的内容（时间戳、用户上下文）下沉到 prompt 末尾以提升复用前缀命中率。
9. **OpenAI Codex 新姿势** — Mac 升级后用 Cmd-Cmd Appshots 把 IDE / 设计稿窗口（含屏幕外内容）直接喂给会话；启用 Remote connection 在 Mac 后台跑 Codex、手机端续接；管理员开启插件分发并接入 Analytics 看板掌握全员用量。
10. **Codex 大仓库回归** — 升级后在大型 monorepo 中重新评估之前因卡顿放弃的工作流（频繁切换线程、长流式输出），把 Codex 启动作为日常开发起点利用「首次交互可用」缩短上下文加载。
11. **Grok 接入 opencode / openclaw** — 在 opencode 中绑定 Grok 或 X Premium 订阅体验 Grok Build 同款模型，与现用编码助手在代码库理解速度上做横向对比。
12. **SynthID 内容检测** — 面向公众发布的图文 / 音视频生成方接入 SynthID 提升可追溯性，在 Gemini App 或 Google Search 中直接询问「是否 AI 生成」对可疑素材二次核验。
