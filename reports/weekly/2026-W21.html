5月18日 – 24日

本周趋势

  • 谷歌集中爆发：Google I/O 同周发布 Gemini 3.5 Flash、Gemini Omni 与 Antigravity 2.0，模型、视频和开发平台一次性更新。
  • 通义冲击前沿：Qwen3.7 Preview 与 Qwen3.7-Max 连续登陆 Arena、百炼和 OpenRouter，阿里开始在 Agent 与编码榜单正面追赶。
  • 编码工具提速：OpenAI Codex 优化大仓库 Git 操作，Claude Code 推出 Fast 模式与缓存诊断，开发者工具竞争转向速度和成本。
  • 企业代理部署：Claude Managed Agents、OpenAI+Dell 与 Codex 远程控制共同显示，Agent 正被带入自托管、本地和移动协作环境。
  • 安全研究前置：Anthropic Glasswing 披露 Mythos Preview 挖出逾万高危漏洞，OpenAI 与 Google 同步强化 C2PA 与 SynthID 溯源。
  • 交互规格重塑：Thariq 与 Anthropic 围绕 HTML、implementation notes 和大型代码库实践发声，长时代理需要更可审阅的规格界面。

Top 10 最重要文章

1. Qwen3.7 Max/Plus Preview 登 Arena：阿里跻身文本第 6、视觉第 5 实验室 ⭐⭐⭐⭐⭐

  • 来源: Qwen (Twitter) | 分类: 模型发布 | 日期: 05/18
  • 标签: Qwen3.7 Arena 阿里巴巴 文本榜 视觉榜

摘要: Qwen3.7-Max-Preview 与 Qwen3.7-Plus-Preview 登陆 LMSys Arena，文本榜和视觉榜都给出强信号：阿里升至文本第 6 实验室、视觉第 5 实验室。Max-Preview 在数学、专家、软件工程和代码分项都进入前列，说明 Qwen3.7 正式版发布前已经具备明显竞争力。对中文和国产模型生态来说，这是一条前沿能力逼近的关键观察点。

2. Google 发布 Gemini Omni：从任意输入生成任意内容的世界模型 ⭐⭐⭐⭐⭐

  • 来源: Google DeepMind (Twitter) | 分类: 模型发布 | 日期: 05/19
  • 标签: Gemini Omni Omni Flash 世界模型 视频生成 多模态 Google Flow

摘要: Google 在 I/O 2026 推出 Gemini Omni，定位为迈向“任意输入生成任意内容”的第一步，并从视频生成和视频编辑起步。Omni Flash 已在 Gemini App、Flow 和 YouTube Shorts 开放体验，支持用照片、视频和音频构建新场景，也支持自然语言改写自有视频。它把 Gemini 的智能与生成式媒体系统合并，成为本周最重要的多模态发布。

3. Gemini 3.5 Flash 发布：前沿智能 + 行动力，速度达同级 4 倍 ⭐⭐⭐⭐⭐

  • 来源: Google AI Blog | 分类: 模型发布 | 日期: 05/19
  • 标签: Gemini 3.5 Flash 编码模型 智能体 Terminal-Bench Gemini API Antigravity

摘要: Gemini 3.5 Flash 是 Google I/O 的模型主轴，主打前沿智能与真实世界行动能力结合。官方称其在 Terminal-Bench 2.1、GDPval-AA、MCP Atlas 和 CharXiv Reasoning 等基准上超过 Gemini 3.1 Pro，输出速度达其他前沿模型 4 倍、成本常不到一半。模型已进入 Gemini App、Search AI Mode、Antigravity 与 Google AI Studio，是低延迟 Agent 和编码任务的重点候选。

4. Google I/O 2026：Gemini 3.5 Flash、Gemini Omni 与 Antigravity 2.0 全发布 ⭐⭐⭐⭐⭐

  • 来源: Google AI Blog | 分类: 模型发布 | 日期: 05/20
  • 标签: Google I/O Gemini 3.5 Flash Gemini Omni Antigravity 智能体

摘要: Google I/O 2026 集中发布上百项更新，把 Gemini 3.5 Flash、Gemini Omni、Antigravity 2.0、Search 信息智能体、Gemini Spark 和 Universal Cart 放在同一套智能体叙事里。Antigravity 2.0 作为开发平台引入 CLI、SDK 和多智能体编排，直接承接 3.5 Flash 的编码能力。相比单个模型发布，这篇汇总展示了 Google 把模型、搜索、购物、开发和媒体生成串成平台生态的意图。

5. Qwen3.7-Max 发布：面向 Agent 时代的新旗舰模型 ⭐⭐⭐⭐⭐

  • 来源: Qwen (Twitter) | 分类: 模型发布 | 日期: 05/21
  • 标签: Qwen3.7-Max 阿里巴巴 智能体 编码模型 长程自主 Artificial Analysis

摘要: Qwen3.7-Max 正式发布，定位为 Agent 时代的通用基座，支持前端原型、多文件重构、真实调试、MCP 集成和多智能体办公流程。模型上下文扩至 100 万 tokens，在 Artificial Analysis 智能指数拿到 56.6 分，并通过自进化实验优化注意力内核取得 10 倍几何平均加速。API 已在阿里云百炼上线，意味着 Qwen3.7 从榜单预热进入真实接入阶段。

6. Anthropic Glasswing 首月战报：Mythos Preview 挖出逾万高危漏洞 ⭐⭐⭐⭐⭐

  • 来源: Anthropic Research | 分类: 研究 | 日期: 05/22
  • 标签: Project Glasswing Claude Mythos Preview 网络安全 漏洞披露 开源安全

摘要: Anthropic 披露 Project Glasswing 首月进展：约 50 家关键基础设施伙伴使用 Claude Mythos Preview 发现逾一万个高/严重级漏洞，Cloudflare 单家上报 2,000 个 Bug。Anthropic 同步扫描 1,000 多个开源项目，确认 1,587 个真阳性漏洞，其中 530 个已披露、75 个已修复。文章最重要的判断是补丁部署速度已成为瓶颈，强模型让漏洞发现和修复周期的治理压力同步上升。

7. Qwen3.7-Max 登陆 OpenRouter：第三方 Agent 测试压制 Opus 4.7 与 GPT-5.5 ⭐⭐⭐⭐⭐

  • 来源: Qwen (Twitter) | 分类: 模型发布 | 日期: 05/22
  • 标签: Qwen3.7-Max OpenRouter Agent 基准 编码模型 Prompt 缓存

摘要: Qwen3.7-Max 登陆 OpenRouter，主打编码、办公生产力和长程自主执行，并支持显式 prompt 缓存。第三方 atomic_chat_hq 的 Tetris Bot 循环任务显示，Qwen3.7-Max 训练成本 $1.32、Bot 性能提升 56%，在该任务上比 Claude Opus 4.7 和 GPT-5.5 都更具性价比。虽然这不是通用结论，但它给 Agent 循环成本评测提供了具体样例。

  • 来源: Demis Hassabis (Twitter) | 分类: 模型发布 | 日期: 05/19

摘要: Demis Hassabis 将 Gemini Omni 描述为世界理解和多模态编辑能力的一次跃升，支持接收照片、视频和音频并构建新场景。Omni Flash 已可在 Gemini App 与 Flow 中试用，用户也能上传自己的视频做创意迭代。它从产品入口角度确认了 Gemini Omni 不是研究演示，而是已经进入创作工具链。

分类概览

模型发布

W21 的模型发布几乎被 Google 与 Qwen 两条线占满。Google 借 I/O 同时推出 Gemini 3.5 Flash 与 Gemini Omni：前者把低延迟、低成本和编码/Agent 基准结合，后者把视频生成、视频编辑和世界模型叙事推到产品端。Qwen 则用 3.7 Preview、3.7-Max 和 OpenRouter 接入连发，既有 Arena 排名，也有长 Agent 循环成本对比。整体看，模型发布不再是单点能力更新，而是模型、工具、API、开发平台和分发生态同步上线。

研究

研究板块最强信号来自 Anthropic Glasswing：Mythos Preview 一个月内协助关键基础设施伙伴发现逾一万个高危/严重漏洞，说明前沿模型已经开始改变漏洞发现的规模和节奏。Google 的 Gemini for Science 与 I/O Dialogues 则把科学发现、量子和机器人讨论纳入 Gemini 生态叙事。研究新闻数量不多，但都指向“AI 系统进入真实科研/安全流程”这个方向。相比模型榜单，真正的瓶颈开始转向验证、补丁部署和实验闭环。

产品与功能

产品与功能是本周最大分类，核心是开发者工具和 Agent 运行环境。OpenAI Codex 做了大仓库性能优化、远程控制、Appshots、团队插件共享，并通过 Dell 进入混合云与本地企业环境。Anthropic 推出 Claude Console 缓存诊断、Claude Code Fast 模式、Managed Agents 自托管沙箱和 MCP 隧道，强调企业可控性与成本优化。Google 则以 Antigravity 2.0、Gemini App、Search AI Mode、Workspace 和 Flow 把 I/O 发布落进多个产品入口。

商业动态

商业动态围绕“渠道扩张”和“企业落地”展开。OpenAI 与 YC、Dell、Khan Academy、纽约时报和新闻出版伙伴继续扩张分发与企业合作，Sam Altman 也强调 AGI 加速科研、企业和个人目标这三条路径。Google 借 I/O 宣布 AI Ultra 订阅、Search 和 Workspace 更新，把模型能力包装成消费与企业套餐。xAI、Runway、Nous 等也在订阅、内容和生态上持续补位，显示前沿能力竞争正在快速进入商业包装阶段。

政策与安全

政策与安全集中在内容溯源和模型/Agent 风险控制。OpenAI 强化 C2PA、SynthID 水印和公开验证工具，Google 也推进 SynthID 与 Content Credentials 生态，AI 生成内容识别开始从单厂商功能走向跨平台标准。Anthropic 与宗教、哲学和伦理学者讨论 AI 道德养成，并通过工具提醒实验降低内部对齐失准行为。Glasswing 的漏洞发现规模也让“安全模型太强后补丁跟不上”成为更现实的治理问题。

教程与观点

教程与观点本周重点是如何与长时代理协作。Anthropic 总结 Claude Code 在大型代码库中的实践，Thariq 则围绕 HTML artifacts、implementation notes 和“脚本 + PDF + HTML”工作流给出一组可操作方法。OpenAI 的 Ramp 案例展示 Codex + GPT-5.5 可以把代码评审反馈从数小时压到数分钟。整体来看，提示词技巧正在从“写得更清楚”变成“给 Agent 一个可持续维护、可审阅、可验证的工作界面”。

本周值得上手试试

  1. Gemini 3.5 Flash — 在 Google AI Studio 或 Antigravity 中接入 3.5 Flash，对比现有编码/Agent 模型在延迟、成本和长链路任务上的表现。
  2. Gemini Omni / Flow — 用自有视频素材测试 Omni Flash 的场景重构、批量编辑和角色一致性，评估是否可进入内容原型流程。
  3. Qwen3.7-Max — 通过阿里云百炼或 OpenRouter 接入 Qwen3.7-Max，在长 Agent 循环和代码任务上对比 Opus / GPT 的成本与质量。
  4. Codex 大仓库性能 — 在 monorepo 中重新跑过去因 Git 操作慢、线程切换卡顿而放弃的 Codex 工作流，量化 10-50 倍提速是否兑现。
  5. Claude Prompt 缓存诊断 — 打开 Claude Console 缓存诊断页，定位未命中缓存的 prompt 片段，并把易变内容下沉到末尾。
  6. Claude Code Fast 模式 — 用 /fast 在现场调试、快速迭代等延迟敏感任务中试跑 Opus 4.7，同时记录 token 成本差异。
  7. Managed Agents 自托管沙箱 — 对合规要求高的内部工具，评估 Claude Managed Agents 的自托管沙箱和 MCP 隧道是否能替代公网暴露。
  8. Antigravity 2.0 — 用任务中控、CLI 和 SDK 编排一个真实多智能体开发任务，观察子代理分工是否比单会话更稳定。
  9. Codex Appshots / 远程控制 — 在 Mac 上用 Appshots 捕获窗口上下文，并从 ChatGPT 手机端接管长任务，验证跨设备协作节奏。
  10. HTML 规格与 implementation notes — 让 Agent 在实现规格时维护 implementation-notes.html，记录模糊决策、偏离原因和待人工确认的问题。
