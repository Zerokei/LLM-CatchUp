## 本周值得上手试试

1. **Claude Managed Agents** — 在 Anthropic 平台申请测试版访问权限，通过 Managed Agents API 部署一个简单的文件处理或数据分析智能体，亲身体验内置沙箱和检查点机制，对比与自建 LangChain/AutoGen 框架的实际差异

2. **Claude Cowork（桌面正式版）** — 升级至任意 Claude 付费计划后安装桌面应用，给 Cowork 一个具体目标（如整理本地文件夹、生成项目周报），观察其自主执行能力；企业用户可进一步测试 RBAC 权限配置

3. **Gemma 4 本地部署** — 通过 Ollama 下载 Gemma 4 26B MoE 版本，在本地测试中文对话、代码生成等典型任务，并与当前使用的 API 模型做横向对比评测

4. **Gemini API Flex 层级** — 若你已在使用 Gemini API，将批量处理任务切换至 Flex 层级，实测成本节省效果；对延迟敏感的在线场景则可测试 Priority 层级的响应速度提升

5. **OpenAI Codex 升级功能** — 在 Codex CLI 中连接一个自定义 MCP 服务器（如本地数据库），并尝试向 Codex 发送截图或线框图以生成对应代码，体验新增的视觉输入能力

6. **Claude for Excel（beta）** — 若有金融或数据分析需求，申请 Claude for Excel beta 访问权限，在侧边栏中尝试构建基础现金流折现模型，感受 AI 辅助财务建模的工作流程

## 本周趋势

**智能体基础设施进入爆发期。** 本周 Anthropic 密集发布 Managed Agents、Cowork 正式版和金融服务扩展，三条产品线共同指向同一战略：将 Claude 从对话工具升级为可信赖的"企业行动层"。OpenAI Codex 的 MCP 扩展和按需计费改革也在同步推进，行业正从"谁的模型更强"转向"谁的智能体基础设施更完善"的竞争维度。

**安全与对齐的张力骤然升温。** "同伴保护"现象研究、Claude Mythos 风险报告、Project Glasswing 以及 Axios 供应链攻击事件，四则消息在同一周爆发绝非巧合——随着模型能力快速突破，安全研究社区对"失控风险"的担忧已从理论走向可测量的实验结论。

**资本与算力双双锁定长期博弈。** OpenAI 1220 亿美元融资与 Anthropic 3.5 吉瓦 TPU 协议标志着 AI 军备竞赛进入重资产阶段。这一量级的资本承诺意味着头部玩家将在未来 2-3 年内在算力上形成难以追赶的壁垒，中小模型厂商面临的竞争压力将持续加剧。

**开源仍有生命力，但窗口期有限。** Gemma 4 以 Apache 2.0 闯入全球前六，说明开源模型的质量已足以与闭源旗舰抗衡。然而随着闭源厂商持续砸入算力和数据，这一窗口期究竟还有多长，将是下半年最值得观察的结构性问题。
